# Uber-Clon-RN

###Uber clon with sandbox for google maps integrations


### Installation steps
- Clone repository.
- Run npm install.
- For running the dev server.

```bash
# Clonning repo
git clone [https://gitlab.com/lupacan/mobile-lupacan.git](https://github.com/roycuenca/Uber-Clon-RN)

cd Uber-Clon-RN
yarn or npm i
 
expo start

```
